SOLR
====
Start `wendy` and:
    (wendy)$ ssh wendy
    $ cd solr-4.6.1/moogle/
    $ java -jar start.jar

If you want to mount Solr's folder locally on your laptop:
    (laptop)$ mkdir /tmp/solr
    $ sshfs wendy:/home/nimiq/solr-4.6.1 /tmp/solr
